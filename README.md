# REE.Unpacker :see_no_evil:
Tool for extract PAK archives from games developed on [RE ENGINE](https://en.wikipedia.org/wiki/RE_Engine)

### Usage
> REE.Unpacker "**TAG_LIST_FILE**" "**INPUT_PAK_FILE**"

> REE.Unpacker "**TAG_LIST_FILE**" "**INPUT_PAK_FILE**" "**UNPACK_DIRECTORY (OPTIONAL)**"

> [!note]
> If the output directory is not specified, then the unpacked files will be in the same place as the PAK file

### Example
> REE.Unpacker **RE4_STM_Release** "**E:\Games\RESIDENT EVIL 4\re_chunk_000.pak.patch_002.pak**"

> REE.Unpacker **RE4_STM_Release** "**E:\Games\RESIDENT EVIL 4\re_chunk_000.pak.patch_002.pak**" "**D:\RE4_Unpacked**"

> [!warning]
> The Projects folder must be in the tool folder, don't forget to copy it :) 

# REE.Packer :see_no_evil:
Tool for creating your own PAK archives

### Usage
> REE.Packer "**INPUT_DIRECTORY**" "**OUTPUT_PAK_FILE**"

### Example
> REE.Packer "**D:\RE4_Unpacked**" "**E:\Games\RESIDENT EVIL 4\re_chunk_000.pak.patch_002.pak**"

> [!important]
> Don't forget to make a backup of the PAK archives

# REE.Fonts.Cryptor :see_no_evil:
Tool for decrypt and encrypt fonts (*.otf.1)

### Usage
> REE.Fonts.Cryptor "**INPUT_FONT_FILE**" "**OUTPUT_FONT_FILE**"

# REE.Rom.Cryptor :see_no_evil:
Tool for decrypt and encrypt CAS and CAS2 ROM files (*.mameac.2)

### Usage
> REE.Rom.Cryptor "**INPUT_MAMEAC_FILE**" "**OUTPUT_MAMEAC_FILE**"

> [!note]
> ROM files are encrypted only on the PC platform.
